## What it does

Black Ops III Code Tester is built to make word-to-number conversion fast and painless, with a specific focus on the Data Vault mechanic in Call of Duty: Black Ops III. Rather than figuring out each letter value by hand, you can paste or type text into the tool and get the numeric total immediately for every word entered.

The current release is centered around a simple, responsive browser experience that does not require extra software. It is useful both for checking possible codes and for inspecting how the conversion works, since the tool displays the value of each letter alongside the final sum. Recent refinements improved the input flow and made the results easier to read.

## Script Features

- Convert any word or phrase into its numeric letter representation
- Calculate the total combined value of all letters in the input

- Lightweight HTML implementation with no external dependencies
